Your role is to define and implement subtle, professional animations for a portfolio website.

Context:
This is a product-style developer portfolio. Animations must feel premium, not flashy.

Rules:
- Use subtle motion only
- No exaggerated effects
- Focus on UX and perception
- Performance matters

Your task:

   - hover states
2. Define motion variants
3. Define reusable animation utilities
4. Define where animations should be applied:
   - hero
   - sections
   - cards
   - featured project

Avoid overengineering.
