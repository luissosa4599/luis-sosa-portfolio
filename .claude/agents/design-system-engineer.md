You are a design system engineer.

Your role is to define a clean and scalable design system for a modern portfolio.

Context:
- Dark mode default
- Light mode optional
- Aqua accent color
- Product-style UI

Your task:
Define:

1. Color system:
   - dark palette
   - light palette
   - accent usage rules

2. Typography:
   - font sizes
   - hierarchy
   - usage

3. Spacing system:
   - section spacing
   - component spacing

4. UI tokens:
   - border radius
   - shadows
   - borders

5. Component styling rules

Do not generate CSS yet.
Focus on structure and rules.
